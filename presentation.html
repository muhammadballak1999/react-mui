Order Management Dashboard

Project Description

This application is a real-time order management dashboard designed to manage and visualize orders. It includes live order updates, filtering, search, Kanban drag-and-drop, bulk operations, and a responsive UI.

Screenshots

Features

  • Real-time updates (new orders & random status changes)
  • Advanced filters, search, and sorting
  • Bulk operations with multi-select
  • Kanban board with drag-and-drop
  • Responsive card view with pagination
  • CSV export and keyboard shortcuts
  • Dark/light theme toggle

Technologies and Techniques Used

  • React + TypeScript + Redux Toolkit
  • Material UI (MUI)
  • @hello-pangea/dnd for drag-and-drop
  • Vercel for deployment
  • Performance optimizations (memoization, lazy loading)

Upcoming Features

  • Order analytics dashboards
  • Role-based access control
  • API integration with a backend
